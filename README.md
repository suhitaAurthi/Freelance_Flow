# **FreelanceFlow** is a role-based Android application that connects clients, freelancers, and teams within a unified freelancing platform. The app provides separate dashboards for each role while using a shared navigation structure to ensure a consistent user experience. Users can register, log in, select roles, manage profiles, and navigate through role-specific features seamlessly. FreelanceFlow emphasizes clean UI separation, modular screen design, and persistent session handling. The project is structured for scalability and maintainability, making it suitable for real-world freelancing workflows.
